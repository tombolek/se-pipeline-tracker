Trade-off: Most information at a glance. Mirrors the existing ConnectionIndicator's dot+text grammar (shown above as "Live"), so the header feels cohesive rather than double-dotted with two separate widgets.

Freshness thresholds: green ≤ 24h · amber ≤ 3d · red > 3d. Tunable per-team once we see usage.

Manager click: clicking the readout jumps to /settings/import-history so they can trigger a new import. SE view is non-clickable, just the tooltip.

Mobile / <768px: text collapses, dot-only shown; tooltip on tap.

Backend implementation

One new endpoint GET /api/v1/opportunities/import/latest (auth-only, not manager-gated). Returns the most recent successful import: { imported_at, filename, row_count, opportunities_added, opportunities_updated }. Polled once on page load + refreshed every 5 min. Source is SELECT ... FROM imports WHERE status = 'success' ORDER BY imported_at DESC LIMIT 1. Offline-aware: cache the last known result in localStorage so the badge still renders without the round-trip.
